# 📌 PythonGame
#### PythonGame은 다양한 술 게임을 파이썬으로 구현한 프로젝트입니다. 각 게임은 팀원들이 맡은 역할에 따라 개발되었으며, 간단한 규칙에 따라 진행됩니다.

# 🎮 게임 목록
1. 369 게임 - 정용화
2. 이구동성 게임 - 김하영
3. 지하철 게임 - 강민주
4. 수도 맞히기 게임 - 정영헌
5. 슈퍼개미 게임 - 이주형

# 🖱️ 실행 방법
```python
pip install BeautifulSoup4
pip install requests
python main.py
```
